# 家电家装家居行业 · 热点挖掘器 — 执行计划

## 项目目标
构建一个按天维度输出的热点挖掘器 Web 应用，帮助直播间运营挖掘全网高热消费热点（天气、节气、节假日、网络热梗），快速组织供给抢占销售机会。

## Stage 1: 深度研究 — 消费热点关联规律
**技能**: `deep-research-swarm`
**目标**: 建立天气/节气/节假日/热梗 → 家电家装家居消费需求的完整映射体系

### 研究维度
- **天气-消费映射**: 研究不同天气模式（梅雨、高温、寒潮、台风等）对应的爆款品类
- **节气-消费映射**: 24节气与家电/家装/家居消费的关联（如立春-换季收纳、立夏-空调电扇等）
- **节假日-消费映射**: 2026年全年节假日消费热点预判（五一装修季、618、双11、年货节等）
- **网络热梗追踪**: 近期家居家电相关热点话题/热梗
- **选品策略库**: 每个热点对应的推荐选品清单

### 输出物
- `/mnt/agents/output/research/hotspot_rules.json` — 完整的消费热点映射规则库
- `/mnt/agents/output/research/2026_calendar.json` — 2026年全年日历（含节气、节假日）

## Stage 2: 产品设计与开发 — 热点挖掘器 Web 应用
**技能**: `vibecoding-webapp-swarm`
**目标**: 构建一个可交互的热点挖掘器 Web 应用

### 核心功能模块
1. **热点日历视图**: 按月/周/天展示热点，支持日期导航
2. **今日热点看板**: 当天天气+节气+节假日+热梗的综合热点提醒
3. **未来7天/30天热点预告**: 运营排期参考
4. **选品推荐引擎**: 每个热点对应的具体选品清单和运营建议
5. **热点强度评级**: 用热度指数直观显示机会大小

### 设计风格
- 运营工具风格，信息密度高但不拥挤
- 低饱和度暖色调为主，重点数据用亮色突出
- 日历视图 + 卡片式热点详情
- 响应式，适配PC大屏和移动端

### 技术栈
- React + TypeScript + Tailwind CSS
- shadcn/ui 组件
- 本地数据存储（研究阶段产出的JSON）

## 输出物
- 完整可部署的 Web 应用
- 部署到线上环境
